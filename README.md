A microlearning music theory app built with Flutter, Spring Boot and MySQL.

## Features

- User authentication (JWT)
- Music theory quizzes
- Progress tracking
- Admin quiz management
- Resource uploads

## Tech Stack

### Backend
- Spring Boot 3 (Java 21)
- Spring Data JPA (Hibernate)
- Spring Security + JWT
- RESTful API design
- Password hashing using BCrypt
- Role-based authorization (Admin / User)

### Frontend
- Flutter
- HTTP API integration
- JWT-based authentication

### Database
- MySQL
- JPA Entity Mapping
- Relational Schema Design

## How to Run

### Backend

1. Create MySQL database
2. Import database/database.sql
3. Create application.properties from example
4. Run Spring Boot

### Frontend

cd frontend
flutter pub get
flutter run
